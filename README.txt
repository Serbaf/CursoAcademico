Implementación de un proyecto Spring Boot con persistencia basada en H2.
La estructura básica y la idea del proyecto están tomadas del curso de YouTube "Spring Boot Quick Start" realizado por JavaBrains,
aunque con pequeñas variaciones sobre el original.

Link: https://www.youtube.com/watch?v=msXL2oDexqw&t=0s&list=PLmbC-xnvykcghOSOJ1ZF6ja3aOgZAgaMO&index=2

El proyecto básicamente implementa lo que podría ser el esqueleto para una base de datos académica con sus materias, cursos y lecciones.
No tiene mayor complicación, pero sirve para aprender el diseño en capas, y sobre todo la utilización de la persistencia con JPA y las
herramientas que proporciona Spring Boot, que permiten incluso omitir la escritura de órdenes SQL.
